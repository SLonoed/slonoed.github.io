--- layout: post title: Сообщение об окончании выполнения bash команды в MacOS categories: ru --- Проблема: вы запустили долгую команду в терминале (например, клонируете огромный репозиторий или компилируете большой пакет) и вам не хочется смотреть и ждать окончания. Можно конечно открыть новую вкладку в терминале и заняться другими делами, иногда проверяя не окончен ли процесс. Решение: использовать terminal-notifier.\ Устанавливаем этот гем: 

sudo gem install terminal-notifier
Теперь можем использовать его следующим образом 

terminal-notifier -title 'Terminal' -message 'Done! You awesome!'
Получаем стандартное сообщение в ОС
Ну и бонусом - удобная команда, которую стоит засунуть в ваш ~/.bash_profile

tm() {
  default="Task done!"
  message=${1-$default}
  terminal-notifier -message "$message" -title "Terminal"
}
Теперь можно использовать следующим образом 

git clone https://github.com/twbs/bootstrap.git && tm #выведет стандартное сообщение по окончании

git clone https://github.com/twbs/bootstrap.git && tm 'Bootstrap repo clone' #выведет кастомное сообщение
