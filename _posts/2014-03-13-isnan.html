--- layout: post title: Кошерный isNaN categories: ru --- Нативный метод isNaN делает приведение типов при проверке. В итоге выражение isNaN('hello') выдаст true. Если необходимо проверить, что переменная является именно константой NaN можно использовать следующий метод. 

function betterIsNaN(v) {
  return v != v;
}
